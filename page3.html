Quick Grammar: There is / There are

Describing Rooms

We use There is for singular items (one thing).

Example: There is a sofa in the living room.

We use There are for plural items (more than one thing).

Example: There are two chairs in the dining room.

Negative: There isn't a... / There aren't any...

Questions: Is there a...? / Are there any...?
