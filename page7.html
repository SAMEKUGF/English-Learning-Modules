Using There Is / There Are

Talking About Existence

We use There is and There are to say that something exists (or doesn't exist) in a place.

  • Use There is for singular countable nouns and uncountable nouns.
  • Use There are for plural countable nouns.

Grammar Structure & Examples

Form Singular Countable / Uncountable Plural Countable
Affirmative (+) There is + a/an + singular noun.
Example: There is a stove in the kitchen.

There is + some + uncountable noun.
Example: There is some milk in the fridge. 		There are + number/some + plural noun.
Example: There are two chairs.
Example: There are some cushions.
Negative (-) There isn't + a/an + singular noun.
Example: There isn't a bed in the kitchen.

There isn't + any + uncountable noun.
Example: There isn't any coffee left.

(Also: There is no + singular/uncountable noun)
Example: There is no pillow. There is no water. 		There aren't + any + plural noun.
Example: There aren't any pillows on the bed.

(Also: There are no + plural noun)
Example: There are no rugs in the dining room.
Yes/No Question (?) Is there + a/an + singular noun?
Example: Is there a lamp in the bedroom?
→ Yes, there is. / No, there isn't.

Is there + any + uncountable noun?
Example: Is there any cheese?
→ Yes, there is. / No, there isn't. 		Are there + any + plural noun?
Example: Are there any chairs in the living room?
→ Yes, there are. / No, there aren't.

Vocabulary Recap

Remember words for rooms (kitchen, bedroom, living room, bathroom) and items found there (sofa, bed, table, fridge, stove, lamp, curtains, etc.).
